About me

Technologies I've used

.NET

Intermediate, since 2019

Most of my apps are made with C#. I'm most experienced working on the frontend with XAML UI frameworks like WPF and Avalonia, although I also endeavor to learn more about the back end of things in the future.

Web development

Beginner, since 2021

This is actually my first website. It is made with plain HTML, CSS, and Javascript, with Jekyll, but I do eventually want to delve deeper into the world of webdev, for better or worse.

UI design

Intermediate, since 2019

I've been making my own GUI apps since 2019 and making UI concepts since before then. Most of my apps are built around SIADL, my design language that aims to strike the right balance between user friendliness and utilitarianism.

Python

Beginner, since 2017

Python was the first programming language I used between 2017 and 2019. I was never particularly proficient at it, even back in the day.

Software I've used

Visual Studio

since 2019, for coding my apps

Adobe XD

since 2018, for creating icons and logos, and, very occasionally, to make UI concepts
