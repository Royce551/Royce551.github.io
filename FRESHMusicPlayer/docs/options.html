FMP has a few settings that you can access by clicking the gear icon on the Navbar.

General

Language - Displays a list of languages to select. Choose "automatic" if you want to use the system language.

Playtime Logging - Whether FMP should keep track of what tracks you've played over time. You can later view the log using FRESHMusicPlayer.Favorites (400KB)

Integration

Discord Rich Presence - Whether FMP will show the playing track and status in your Discord presence

System Transport Controls - Whether FMP will integrate with Windows 10. This lets you control FMP with the media keys on your keyboard and see what you're playing in the lock screen, for example. No effect on Windows 8.1 or older.

Updates

Auto Update Mode - How FMP will deal with updates:

  • Prompt before updating - Updates will be automatically downloaded, but you'll be asked to restart FMP.
  • Automatic - Once the update has been downloaded and installed, FMP will restart automatically.
  • Manual - FMP won't check for updates automatically; you'll have to press the check for updates button yourself.

Since FMP updates are tiny (< 100KB), you should probably leave this on Prompt.

Appearance

Show time instead of title in window title - Whether the track position will be shown in the window title instead of the track title. This can be useful if you have taskbar labels enabled.

Theme - The color theme FMP will use.

Maintenance

Reset settings to default - Resets all settings to defaults.

Clean and update library - Updates metadata for all tracks in the library and removes duplicate tracks. Useful if you edited metadata using an external tool.

Nuke library - Removes all tracks in the library.
